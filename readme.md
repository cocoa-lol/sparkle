# Installation
Run install.py with an active internet connection to install Sparkle, and add Sparkle to your System path with the instructions below.

# Windows
1. Add . to the local user path (Ex: C:\Users\{user}\Documents\sparkle)
2. Run the command 'sparkle'

# Linux - MacOS
1. Copy sparkle.py to ~/.local/bin/ and rename sparkle.py to sparkle
```bash 
mv sparkle.py sparkle && mv sparkle ~/.local/bin/sparkle
```
